# Palette View

This is the view of your palette. This view is simple but also very complete.

## Move a color

Each color in the palette has a Move button to modify its position in the palette by dragging and dropping to the desired location.

## Color information

Each color in the palette has a primary information, which is the color in hex format, and a secondary information, which is the color name _(which is editable in the tool settings)_.

### Shades, Tints and Tones

<p
  style={{
    display: 'flex',
    alignItems: 'center',
  }}
>
  The button is represented by the icon
  <TableCellsIcon
    style={{
      width: '1.25rem',
      marginLeft: '0.35rem',
    }}
  />.
</p>

In color theory, a **tint** is a mixture of a color with white, which increases lightness, while a **shade** is a mixture with black, which increases darkness. Both processes affect the resulting color mixture's relative saturation. A **tone** is produced either by mixing a color with gray, or by both tinting and shading.

## Add new color

Passing the mouse between two colors you will see a plus button appear allowing you to create 1 or more colors which will be a mix between the two colors.

<img
  alt="Add Color"
  src="/img/palette-creator/color-add.png"
  style={{
    width: '100%',
    maxWidth: '50px',
    borderRadius: '5px',
    boxShadow: '0 0 10px rgba(0, 0, 0, 0.1)',
    marginBottom: '0.5rem',
  }}
/>

For colors located at the ends of the palette, you will see the plus button appear hovering the end of the color. This button works the same way except that it will not create a color mix, but shades of your color.
