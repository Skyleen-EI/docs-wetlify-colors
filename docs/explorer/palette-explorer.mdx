## View a color

You can hover your mouse over each color in a palette to get its name and hex code.

You will also see a menu appear with 2 buttons:

- A <HeartIcon style={{
    width: '1rem',
    height: '1rem',
    marginX: '0.25rem',
    verticalAlign: 'middle',
    marginBottom: '0.25rem',
  }} /> button to save the color.
- A <InformationCircleIcon style={{
      width: '1rem',
      height: '1rem',
      marginX: '0.25rem',
      verticalAlign: 'middle',
      marginBottom: '0.25rem',
    }} /> button to view the color in detail.

## View a palette

You can click on the <EllipsisHorizontalIcon style={{
    width: '1rem',
    height: '1rem',
    marginX: '0.25rem',
    verticalAlign: 'middle',
    marginBottom: '0.25rem',
  }} /> button, located below each palette, to open a menu with 4 buttons:

- **Save**: Save the palette.
- **Open**: Open the palette in the Palette Creator tool.
- **Information**: Open the palette detail page. There you will find all the color information in the palette as well as the **Copy**, **Open** and **Save** buttons in the **Options** dropdown.

  <img
    alt="Palette detail"
    src="/img/explorer/palette-detail.png"
    style={{
      width: '100%',
      maxWidth: '500px',
      borderRadius: '5px',
      boxShadow: '0 0 10px rgba(0, 0, 0, 0.1)',
      marginBottom: '0.5rem',
    }}
  />

- **Copy**: Copy the palette links to the Palette Creator tool.
