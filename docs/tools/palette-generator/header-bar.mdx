# Header Bar

The header bar is composed of the following elements:

- **Accueil** button: Go back to the home page

- **File** button: Open an action menu

- **Generate** button: Generate a random palette _(taking into account the locked colors and the generation method chosen in the settings)_

- **Undo** and **Redo** buttons: Undo or redo the last action

- **Title** of the palette: Click to edit the title

The file menu is composed of the following elements:

- **Title** of the palette: Click to edit the title

- **Create new**: Create a new palette

- **Generate** button: Generate a random palette

- **Save as**: Save the palette as a new palette in your project _(only available if the palette has not already been saved)_

- **Save**: Update the saved palette _(only available if the palette has already been saved)_

- **Export**: Export the Palette

## Palette exporter

  alt="File menu"

You can export the palette in the following formats:

- **URL**: A URL that can be shared with anyone _(the palette is encoded in the URL)_

- **Code**: The palette in different code formats: Default, CSV, Array, Object and JSON

- **CSS**: The palette in Hex, RGB and HSL format for CSS

- **SCSS**: The palette in Hex, RGB and HSL format for SCSS

- **TailwindCSS**: The palette in Hex and RGB format for TailwindCSS

- **Image**: Export the palette as an image in SVG, PNG, JPG or WEBP format with the export builder

- **PDF**: Export the palette as a PDF with the export builder

- **Embed**: Embed the palette in your website
  <img
    alt="Embed"
    src="/img/palette-creator/palette-embed.png"
    style={{
      width: '100%',
      maxWidth: '700px',

      marginBottom: '0.5rem',
    }}
  />
- **ASE**: Export the palette in ASE format for Adobe software

- **Facebook**: Share the palette on Facebook

- **Twitter**: Share the palette on Twitter

- **Pinterest**: Share the palette on Pinterest
