# Color Mixer

Go in the [Color Mixer](https://colors.wetlify.com/color-mixer) tool.

The Color Mixer allows you to mix between 2 and 6 colors to give a unique color that matches your needs.

## How to use it

It is composed of two parts.

At the top you will see your mixed color.

At the bottom you will have a bar containing the different colors that create your mixed color.

### Add a color

To add a color, click on the `+` button.

A new random color will appear on the far right of the bar.

### Copy a color

To copy a color, click on the hex code of the color you want to copy.

### Edit percentage

You will find a bar between two colors that you can move to adjust the color percentage.

### Edit, Save and Delete a color

By hovering your mouse over a color in the bottom bar, you will see a menu appear containing 3 buttons:

- A <EyeDropperIcon style={{
    width: '1rem',
    height: '1rem',
    marginX: '0.25rem',
    verticalAlign: 'middle',
    marginBottom: '0.25rem',
  }} /> button to edit the color
- A <HeartIcon style={{
    width: '1rem',
    height: '1rem',
    marginX: '0.25rem',
    verticalAlign: 'middle',
    marginBottom: '0.25rem',
  }} /> button to save the color
- A <XMarkIcon style={{
    width: '1rem',
    height: '1rem',
    marginX: '0.25rem',
    verticalAlign: 'middle',
    marginBottom: '0.25rem',
  }} /> button to delete the color
